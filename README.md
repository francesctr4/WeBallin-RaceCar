## Game Controls

### KEYBOARD ###
	
	W --> Move Forward
	A --> Move Left
	S --> Move Back
	D --> Move Right
	
	SPACE --> Jump
	LEFT SHIFT --> Boost
	
	UP --> Rotate Forward
	DOWN --> Rotate Back
	LEFT --> Rotate to the Left
	RIGHT --> Rotate to the Right

	ESC --> Exit Game
	
### DEBUG KEYS ###

	F1 --> Toogle Show Colliders
	F2 --> Toogle Camera Control
	F3 --> Restart Level
